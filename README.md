# 🛒 Website Bán Hàng Online

## 📌 Giới thiệu
Website bán hàng online được xây dựng nhằm hỗ trợ người dùng dễ dàng mua sắm các sản phẩm trực tuyến.  
Dự án cung cấp giao diện thân thiện, tốc độ nhanh và tích hợp nhiều tính năng cần thiết cho một cửa hàng thương mại điện tử.

## 🚀 Tính năng chính
- Đăng ký/Đăng nhập tài khoản người dùng
- Tìm kiếm sản phẩm theo tên, danh mục
- Giỏ hàng: thêm, xóa, cập nhật số lượng sản phẩm
- Thanh toán trực tuyến (tích hợp cổng thanh toán)
- Quản lý đơn hàng cho khách hàng và admin
- Trang quản trị: thêm/sửa/xóa sản phẩm, quản lý người dùng
- Responsive UI: hỗ trợ trên máy tính, tablet, điện thoại

## 🛠️ Công nghệ sử dụng
- **Frontend**: HTML, CSS, JavaScript, [React/Vue/Angular] (tùy chọn)
- **Backend**: Node.js (Express) hoặc PHP (Laravel)
- **Cơ sở dữ liệu**: MySQL / MongoDB
- **Triển khai**: Heroku, Vercel hoặc Hosting riêng
